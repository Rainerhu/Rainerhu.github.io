安装前的准备:

step1:

Docker需要一个64位的安装，无论你的Debian版本。 此外，你的内核必须至少是3.10。 最新的3.10次要版本或更新的维护版本也是可以接受的

step2:

对于Debian wheezy用户i必须提供反向端口，启用backports

  • 用任意编辑器打开/etc/apt/sources.list.d/backports.list文件，如果没有这个文件，就创建一个。
  • 删除现有所有条目后添加一条新的反向端口条目

step3:

更新软件包信息

更新存储库信息

添加新的GPG key（这里可能会失败几次，提示从公钥服务器接受失败，多试几次就可以了）

用任意编辑器打开/etc/apt/sources.list.d/docker.list文件，如果不存在，就创建一个
删除现有的所有条目，并添加新的条目（我只列出wheezy添加的，其他版本的请自行到官网查看）

更新apt存储库，并验证apt存储库是正确的

安装包的组成

2014年7月7日，CentOS官方发布了首个CentOS7正式版。CentOS7的Linux内核升级到3.10.0，对Linux容器支持得到大大增强。

在CentOS7的Extra仓库中，有Docker的发布版。这个发布版包含了docker的一个比较老的版本(0.11.1)。相信随着Docker稳定版本的逐步退出，CentOS7会升级源仓库中的docker版本。
